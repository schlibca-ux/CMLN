의학도서관 공지사항

#1. 의학도서관 원문복사 서비스

꼭 필요한 논문인데, 원문 다운로드가 유료일 때? 의학도서관 원문복사 서비스를 이용해보세요!


📝 개요

- 내용: 의학도서관 미구독 학술지 및 유료로 제공되는 국내외 논문의 원문을 대신 구해드리는 서비스

- 대상: 순천향 천안병원 교직원

- 비용: 도서관 예산으로 지원되며 이용자 부담 없음


🔎 상황별 신청 방법 안내

Ⅰ. 홈페이지에서 직접 신청⭐⭐⭐

- 절차: 의학도서관 홈페이지 접속 및 로그인 → 학술연구지원 → 원문복사

- 홈페이지 신청은 1건씩 가능하며, 2건 이상은 문헌 정보와 소속 부서를 기입하여 의학도서관 이메일로 신청(📧 schlibca@schmc.ac.kr)


Ⅱ. PubMed, Discovery 검색에서 바로 신청

ⅰ. PubMed 경유

- 절차: 의학도서관 홈페이지 접속 및 로그인 → 메인 화면 최하단에서 'PubMed' 배너 클릭하여 접속 → PubMed에서 검색한 논문 클릭 → 우측 상단 'SCH 의학도서관 원문복사' 배너 클릭 → '원문복사신청' 클릭 후 신청자 정보 상세 기입(소속부서까지 작성)

* '원문복사신청'이 안보이는 자료는 바로 원문 확인 및 다운로드가 가능한 것임


ⅱ. Discovery(통합검색) 경유

- 절차: 의학도서관 홈페이지 접속 및 로그인 → 메인 검색창에서 논문 검색 → 원문 미보유 논문인 경우 'Full Text Finder' 클릭 → '원문복사신청' 클릭 후 신청자 정보 상세 기입(소속부서까지 작성)


📌 참고사항

- 원문 확인 및 다운로드를 위해서는 프록시 연결을 통한 'PubMed' 접속이 필요합니다. (의학도서관 홈페이지 로그인 필수)

   →  계정 신청 바로가기

- 문헌정보는 자세할수록 빠르게 처리됩니다. (논문 제목, 저자, 저널명, 연도, DOI/PMID 등)

- 여러 편의 논문을 한 번에 신청하실 수 있습니다. (📧 schlibca@schmc.ac.kr)

- 원문 제공까지 업무일 기준 국내 자료의 경우 1~3일, 해외 자료는 1~30일까지 소요될 수 있습니다.
